JavaScript

本笔记仅记录一些需要强加记忆之处和重点.

变量

1.变量命名规则

  • 小驼峰法
    variableName
  • 大驼峰法
    VariableName
  • 下划线法
    variable_name

2.typeOf()

获取变量类型
注意:它不能用来判断是 Array 还是 Object.
可以使用Array.isArray(元素)或元素 instanceof Array

3.变量声明

  • 可以重复声明变量,但只有重新赋值可以改变变量的值,声明不会.
  • 声明时可以用 new 来专门声明类型.但这并不会影响之后的类型转换.
  • 倘若一个变量未被声明,在非严格模式下它仍可运行,会成为 window 的属性之一,属于全局变量的可配置属性,可删除.

4.js 对象

寻址方式

  • 变量名.属性
  • 变量名['属性']

键值对

  • 键必须是字符串/符号.

5.变量比较

等于号

  • ==
    倘若类型不同,会转化为同一类型看值是否相等.
  • === 类型不同便不等.

条件运算符

variablename=(condition)?value1:value2
value1:条件真,value2:条件假.

6.清空变量

  • 变量 = undefined 恢复到未赋值变量状态
  • 变量 = null 释放指向对象的引用.

7.声明提升

  • js会默认将函数及变量的声明提到最上面,因此可以先使用,再声明.
  • 但是初始化,即x=5的变量将不会提升.

函数

1.箭头函数

(函数的参数) => {函数体};
+ 倘若参数只有一个,可以省略包裹参数的括号. + 没有参数-->空括号

  • 字符串内需要添加反斜杠(某些浏览器可能不允许)
  • 使用字符串加法.

JavaScript

初识 JavaScript

1.编程语言和标记语言的区别

编程语言 标记语言
很强的逻辑/行动能力 不发出指令
主动 被动读取


2.介绍

  • 运行在客户端的语言.
  • 是脚本语言:不需要编译,逐行解释执行.
  • 也可以基于 Node.js 进行服务器编程.

3.浏览器执行方式

浏览器包括:

  • 渲染引擎/内核
  • Js 引擎/Js 解释器

4.Js 组成

  • ECMAScript
    Js 语法
  • DOM
    页面文档对象模型-->对元素进行操作
  • BOM
    浏览器对象模型-->对浏览器窗口操作

5.书写位置

  • 行内
  • 内嵌
  • 外部

行内

<input type="button" value="点击我" onclick="alert('嘿嘿');">

内嵌

<script>
    alert("xxx");
</script>

外部

引用: <script src="xxxx.js"></script>

注意!这是个双标签,但是中间不能写代码~

7.Js 输入输出语句

变量

1.声明方式

var 变量名称; //单个
var 变量1=xxx,变量2,等等  //多个
变量=xxx //不声明直接用也可以,不推荐.

2.命名规范

  • 不能以数字开头
  • 严格区分大小写
  • 只允许$和下划线在变量中
  • 驼峰命名法:
    首字母小写,后面每一个单词首字母大写.

数据类型

运行过程中,根据赋值判断数据类型.
js 拥有动态类型,变量数据类型可以变化.

1.简单数据类型

  • Number 数字型
    包含整型和浮点.
  • Boolean 布尔型
  • String 字符串
  • Undefined
    声明了但没赋值.
  • Null 空值

数字型

  • 前面加 0 : 8 进制
  • 加 0x : 16 进制
  • 无穷大: Infinity
  • 无穷小: -Infinity
  • 超过数字型的范围便认为是无穷.
  • 非数字: NaN
    得不出想要结果,不是数字时.
  • 判断是不是数字: isNaN()
    如果是数字,返回 false.

字符串型

1.引号 如果发生引号嵌套,为了避免歧义,需要外面用双引号,内部用单引号.
引号匹配时遵循就近原则.

2.转义符

  • \n 换行 [n:newline]
  • \ 斜杠\
  • \' 单引号
  • \" 双引号
  • \t tab 缩进
  • \b 空格(blank)

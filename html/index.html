Html

骨架结构

Html标签往往成对出现,分别称作开放标签和闭合标签. 每个网页都有一些基础结构标签:

<!DOCTYPE html>  
<html lang="en">
    <head>  
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    </head>
    <body> 
    </body>
</html>

让我们学习他们的含义:

  • <!DOCTYPE html> 通用声明,告诉浏览器这是Html5.[DOC:文档,TYPE:类型]
  • <html lang="en"> [lang:language,语言],它说明本网页的语言格式
  • <head> 头部标签
    <title> 文档的标题.
  • <body> 主体标签,我们的大多数元素都放在里面.

段落标签

还记得初中学的单词paragraph吗?他的意思就是段落.

这是一个单标签,它并不成对,你可以不加/,但是这样可以让代码更加规范.

3.图像标签

<img src="图片地址" alt="消失后的语句" />

每个img标签必须有路径:

  • 相对路径:
    同一级文件:picture.png (直接输入文件名即可)
    下一级文件:pic/picture.png (进入pic后才可读取)
    上一级文件:../pic/picture.png (先到上一级文件夹,再进入pic后才可读取)

  • 绝对路径:(指绝对位置,而不会因html文件位置而改变.)
    网上的图片:http://baidu.com/......
    本地路径:F:\WEB\Note\pic
    我们习惯用\表示盘符开始的绝对路径.

图像标签的属性

  • src 图片路径
  • alt 消失后的替换文本
  • title 提示文本,将鼠标放于其上显示的文字
  • width 宽度
  • height 高度
  • border 边框粗细 他们统一写在img标签内部,使用空格分开,用等号赋值.

注意!单独修改宽或高,浏览器会自动调整图片大小,而不会失真.

表格，列表与表单

1.表格

基本标签语法

  • <table></table> 表格定义标签
  • <tr></tr> 表格的行
  • <td></td> 表格中的单元格
  • <th></th> 表头单元格

注意：

  • 别想不劳而获，一级一级往下嵌套，都得出现才可以使用~
  • 表头默认加粗，居中。
  • 表格里可以放任何元素。

基本标签属性

  • align {left,center,right} 对齐方式。 [align:对齐]
  • border {1/“”} 有无边框。 （表格外部）
  • cellpadding {像素值} 单元格与内容的空隙。 （单元格内部）
  • cellspacing {像素值} 单元格之间空白 （单元格外部）
  • width {像素值} 表格宽度
  • height {像素值} 表格高度
    注意：写在标签内部！

表格结构标签

表格可以分为头部区域与主体区域：

  • <thead> 头部区域
  • <tbody> 主体区域

挺像html的结构，对吧？前面加个t（table）就好了。

合并单元格

  • rowspan {=“合并几个”} 跨行合并 [row:行]
  • colspan {=“合并几个”} 跨列合并 [col:列]

要点：

  • 合并时，以靠左上角的单元格为基点，然后定义合并几个.
  • 必须写在单元格元素中（td/th）
  • 合并了之后记得删掉除了基点以外其他被合并的单元格。
  • 千万别搞混哦~ colspan将横向的单元格合并，rowspan时纵向。因为span就是跨越，跨越了它，不正说明与它垂直么？

表单

组成：

  • 表单域 表单元素所在的整个一大块。

表单域

表单域标签：

表单域的属性：

  • name和value每个都有
